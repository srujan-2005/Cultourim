# Cultourium

A unified digital platform for exploring culture and tourism.

## Features

- AI Chatbot
- Map Exploration
- Heritage Discovery
- Budget Planner

## Installation

```bash
git clone https://github.com/srujan-2005/Cultourim.git
cd Cultourim/cultourim
npm install
```

## Running

```bash
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Google Generative AI
- Leaflet Maps
